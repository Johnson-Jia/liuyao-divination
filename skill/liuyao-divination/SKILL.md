# 六爻占卜断卦 · Liu Yao Divination

## Overview

六爻是以《周易》六十四卦为本、结合**月建日辰**对六爻逐爻分析的传统占卜法。核心原则一句话：

> **先排准卦盘（机械层），再做需要推理的判断（解读层）。** 排盘交给引擎或严格按规则手工执行；解读围绕「用神」展开，以月日定旺衰，以动静/空破/化变/合墓定吉凶与应期。

## When to Use

**用：** 用户提供卦象（卦名/动爻/月日，或三枚铜钱爻象）要求断卦、断吉凶、断旺衰、断某事（姻缘/财运/事业/失物/健康…）；或要求解释某卦盘、某技法。

**不用：** 紫微斗数、八字命理、奇门遁甲、梅花易数（不同体系，技法不通）；纯哲学层面讨论《易经》卦辞（那是经学，不是六爻纳甲断法）。

## Interpretation Workflow — 断卦八步

```
排盘定盘 → 取用神 → 用神旺衰 → 世爻状态 → 查动静暗破 → 查化变合墓 → 综合定性 → 定应期
```

### 第 1 步：取用神（最重要）
用神是全卦核心，取错则全错。常用取法见下「用神速查」。**用神临应爻**时，用神与应爻重叠，指向明确。

### 第 2 步：用神旺衰（看月日）
- **月建**定一月内旺衰，**日辰**定当日生克冲合。两者同权。
- 旺衰看两层：①五行旺相休囚死（月令）；②十二长生（长生/帝旺/墓/绝…）。
- 月生日比和为旺；月克日冲为衰。**旺衰决定用神"有没有力"，不决定"吉凶方向"**——方向看生克关系。

### 第 3 步：世爻状态（求测者自身）
看世爻的六亲、旺衰、是否入墓/空/破、是否发动化进化退。世爻是"你自己"，世爻退弱则自身不济。

### 第 4 步：查动静、暗动、空亡、月破、日破
- **动爻**：明动，力量显，主变化。
- **暗动**：静爻被日辰冲，**旺相**则暗动（激发起用、有用）；**休囚**则日破（冲散无用）。⭐ 最易判错，详见关键辨析。
- **空亡**：旬空之爻，旺不为真空、衰则空。动而化空、空而逢冲各有解法。**日月建本身不以空论。**
- **月破**：月建所冲之爻（如未月冲丑）。
- **日破**：日辰所冲之爻（如卯日冲酉）——但若该爻旺相，则为**暗动**而非日破。

### 第 5 步：查化进化退、变爻
动爻之变：化进化退（同五行地支进退）、化空化破、化回头生克等。化进神主增益前进，化退神主减退退缩。

### 第 6 步：查三合六合、墓库
- **三合**：申子辰/亥卯未/寅午戌/巳酉丑，局成则合力大。
- **入墓**：爻入五行墓库（木墓未、水墓辰、火墓戌、金墓丑、土墓辰）。入墓主困束、暂不出头；**墓库待冲方出墓**。

### 第 7 步：综合定性
合参：用神与世爻的生克关系（用神生世=吉/旺我，克世=耗损）、各方旺衰、动静空破合墓、主变卦卦意。给出**有层次、有条件**的结论，不要一句吉凶了事。

### 第 8 步：定应期
- 用神/世爻**衰而待令**：等当令之月（如财爻亥水衰，待亥子月旺）。
- 入墓**待冲出墓**：如入未墓，待丑月（丑冲未）。
- 空亡**待填实/冲实**。
- 卦辞提示周期（如复卦"七日来复"）。

## Critical Distinctions — 关键辨析（易错处）

### ⭐ 1. 暗动 vs 日破（最高频错误）
静爻被日辰冲：
- **旺相**（月生日比和、得令）→ **暗动**：被激发，能生克他爻，**有用**，主暗中有人相助/暗中发力。
- **休囚衰弱**（月克日泄）→ **日破**：本弱又被冲散，**无用**。

> 例：子孙酉金在未月（未土生金，金旺相），逢卯日冲 → **暗动**（生财），**不是日破**。判错则结论相反。

### 2. 「贵」≠「旺」
爻临五爻君位（九五之尊）是**地位尊贵、显要、被重视**（贵）；但力量强弱（旺）仍看月日生克。**临君位而受月日制压 = 贵而暂弱**，不可因"位高"就断"力强"。

### 3. 空墓 / 出墓
- 入**空墓**（墓库地支恰落旬空）：墓不牢，困束是虚的、易脱，非死局。
- 出墓靠**冲墓**（如未墓待丑冲），不是靠时间硬等。

### 4. 进退神方向（同五行地支）
- 进神：寅→卯、巳→午、申→酉、亥→子、丑→辰→未→戌
- 退神：卯→寅、午→巳、酉→申、子→亥、戌→未→辰→丑
- 化进主增益前进，化退主减退退缩。

### 5. 元神 / 忌神暗动
- **元神**（生用神者）暗动 → 暗中生助用神，**吉**。
- **忌神**（克用神者）暗动 → 暗中克损耗用神，**凶**。
- 判暗动时务必先定该爻是元神还是忌神。

### 6. 月破 vs 日破
- 月破 = 月建所冲（管一月）。
- 日破 = 日辰所冲（当日）；但旺相逢日冲是暗动非破。

### 7. ⭐ 三合"虚合待用" vs 实成之局
三合论实局须**三爻齐动 / 两动一静 / 内外卦动变凑局**。**单爻独动、仅借月日凑合者，只是"虚一待用之合"**——可作牵绊、有合意向的辅助参考，**不可断"局已成、合化生世/克世"的决定性旺力**，须待流月填实方发力。最易把虚合放大为实局的合力。

### 8. 冲墓的双向（应期辩证）
冲墓出墓利"入墓之爻"，但**冲墓之支可能同时克泄用神**（如冲未墓用丑，丑土克水则财亥水转弱）。应期当取"用神得令 ∩ 墓可冲"的交集，勿只看出墓利好。墓库配属勿混：**水墓辰、金墓丑**（不是丑为水墓）。

## Common Mistakes — 常见错误

| 错误 | 正解 |
|---|---|
| 旺相逢日冲判为日破 | 旺相逢冲是**暗动**，休囚逢冲才是日破 |
| 五爻君位 = 力量最强 | 君位是**贵**（地位），旺衰仍看月日 |
| 入墓一律断凶 | 入**空墓**易脱；墓待冲出墓，未必凶 |
| 把月建当空亡论 | **日月建不以空论**；只看卦中爻是否落空 |
| 未土对木说成"生扶助旺" | 土不生木（木克土）；未对木是**墓库收藏**，非生扶 |
| 化退神单一归因 | 化退主退意，但成因是多象（官鬼+入墓+化退…），勿单因 |
| 一句吉凶了事 | 应给层次化、有条件的结论 + 应期 |
| 卦中无空亡爻却大谈空亡 | 先查卦中爻是否落旬空，无则空亡不构成主要矛盾 |
