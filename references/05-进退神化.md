# 05 · 进退神化

动爻必变，变出之爻（变爻/化神）与动爻的关系，决定了动爻力量的增减与事情的进退走向。

## 一、化进神 / 化退神（同五行地支）

动爻与变爻为**同五行**的地支时，论进退神（看地支在该五行序列中的进退方向）。

| 五行 | 进神（前进） | 退神（后退） |
|---|---|---|
| 木 | 寅 → 卯 | 卯 → 寅 |
| 火 | 巳 → 午 | 午 → 巳 |
| 金 | 申 → 酉 | 酉 → 申 |
| 水 | 亥 → 子 | 子 → 亥 |
| 土 | 丑 → 辰 → 未 → 戌 | 戌 → 未 → 辰 → 丑 |

（土按四库顺序，丑→辰→未→戌为进，反之为退。）

- **化进神**：力量增益、前进、发展、壮大、越动越好。
- **化退神**：力量减退、后退、退缩、衰减、越动越弱，主退意、放弃、势微。

> **经典例**：世爻丁卯木动，化庚寅木。卯→寅为**退神**。世爻动而化退，主自身力量减退、心气在退、有打退堂鼓之意。

**进退神只论同五行地支。** 若动爻与变爻五行不同（如木变金），不论进退，而论回头生克（见下）。

## 二、化回头生 / 化回头克

变爻（化神）回头作用于动爻本身：

| 关系 | 名称 | 含义 |
|---|---|---|
| 变爻**生**动爻 | 化回头生 | 得生扶，力量增强，吉（用神化回头生尤吉） |
| 变爻**克**动爻 | 化回头克 | 受克制，力量受损，凶（用神化回头克主事败） |
| 变爻与动爻**同五行但非进退**（如寅化寅） | 化比和 | 力量维持，无明显增减 |

> 例：寅木动化酉金——金克木，为**化回头克**，木爻受损，主事情反而变糟。
> 例：寅木动化子水——水生木，为**化回头生**，木爻得生扶，主事情得助。

## 三、化空 / 化破 / 化墓

变爻本身的状态也会影响动爻的成败：

- **化空**：变爻落旬空。动而化空，主事将有变却落空、暂时不能落实，待变爻填实（出空之日/月）方成。用神化空多主虚象、暂不成。
- **化破**：变爻为月破或日破。动而化破，主变出之物无用、事将变而复败。
- **化墓**：变爻入墓（见 07）。动而化墓，主事被收藏困束，待冲墓方出。

> 三者共同含义：**动爻虽动，但变爻本身残缺/落空/被困，则变化不能落实**，事情多中途有变或暂不能成，须待变爻得救（填实/出破/出墓）。

## 四、动爻变化的实务判读

排盘后看每个动爻，依次问：

1. 变爻与动爻**同五行**吗？
   - 是 → 论**进神/退神**（增力或减力）。
   - 否 → 论**回头生/回头克/比和**。
2. 变爻**本身空、破、入墓**吗？
   - 是 → 变化不能落实，待救应。
3. 综合：动爻是**越动越好**（化进/化回头生/变爻旺实）还是**越动越差**（化退/化回头克/化空破墓）。

## 五、动爻吉凶速查

| 动爻变化 | 趋势 |
|---|---|
| 化进神 / 化回头生 / 化旺 | 越动越好，增益、前进 |
| 化退神 / 化回头克 / 化衰 | 越动越差，减退、受损 |
| 化空 / 化破 / 化墓 | 变化不能落实，暂不成，待救 |
| 化比和 | 维持现状 |
