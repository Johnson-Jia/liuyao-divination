# 06 · 三合六合

"合"主和合、牵绊、聚集。合能把分散的力量聚成一股（三合成局），也能把爻绊住不动（合住）。合与冲相对：冲主动荡散开，合主聚集纠缠。

## 一、三合局

三合局由三个地支合成一股五行之力：

| 三合 | 化局 | 中神 |
|---|---|---|
| 申 · 子 · 辰 | 水局 | 子 |
| 亥 · 卯 · 未 | 木局 | 卯 |
| 寅 · 午 · 戌 | 火局 | 午 |
| 巳 · 酉 · 丑 | 金局 | 酉 |

（每局由"长生·帝旺·墓库"三支组成，中神为帝旺之支。）

### 成局条件
1. **三字齐备**：可由卦中爻 + 月建 + 日辰凑成。
2. **中神旺或有动**：中神（子/卯/午/酉）须旺相或发动，否则合而不化、力弱。
3. **无破解**：三字不被冲破太过，否则合局散。

### 三合局的含义
- 合力**大而专一**，化出之五行力量强，相当于一个强大的该五行之爻。
- 主**聚集、合伙、牵绊、纠葛**——事情不会轻易散开，多方力量拧成一股。
- 合局之五行若为用神所需则大吉（如用神水，得申子辰水局大助）；若为忌神则大凶。
- 合局也主**缠绵、拖延**（合住之意），事情纠缠不清。

> **经典例**：亥（用神妻财水？否，亥卯未化木）— 实例：用神亥水 + 世爻卯木 + 月建未土，凑成**亥卯未三合木局**。木之"生(亥)、旺(卯)、库(未)"齐聚，化为木局（官鬼），合力聚于世爻——主缘分有牵绊、不轻散，但化官鬼带压力纠缠。

> 注：本例中三合化木（官鬼），与用神亥水本属水不同——合化后以**化出之五行**论局之力量，原各支的六亲属性退居其次。这是三合的特点：化而为一。

## 二、六合（地支六合）

相邻地支两两相合：

| 六合 | 化气（参考） |
|---|---|
| 子 · 丑 | 土 |
| 寅 · 亥 | 木 |
| 卯 · 戌 | 火 |
| 辰 · 酉 | 金 |
| 巳 · 申 | 水 |
| 午 · 未 | 火（或土） |

### 六合的两种作用
1. **合住（绊）**：爻被合住则**不能动**，主拖延、羁绊、停顿、纠缠不清。动爻被合则不能动弹（须冲开方动）。
2. **合起（助）**：衰爻逢合得助，或合化为旺相之五行而增力。

### 六合含义
- 主**和合、亲近、牵连**。
- 占姻缘见合为吉（主和合亲近）；占谋事见合住则主拖延不动。
- 用神与世爻相合 → 双方亲近、缘分深。
- 忌神与用神相合 → 忌神被绊住，暂不克用神，反吉（但合解则克复发）。

## 三、合与冲的实务

| | 主 | 含义 |
|---|---|---|
| 三合 | 聚集化一 | 合力大、专一、牵绊、合伙 |
| 六合 | 两两和合 | 亲近、绊住、拖延 |
| 冲（六冲） | 散开动荡 | 变动、冲突、散、破 |

- **合逢冲则散**：合局/六合遇冲则解开（合住之爻遇冲则能动了；合局遇冲则散）。
- **冲逢合则止**：相冲之爻遇合则冲突暂止。
- 占事喜合忌冲（多数情况下，合主成、冲主变），但**合未必皆吉**（合住用神反主迟滞），**冲未必皆凶**（冲开墓库、冲散忌神反吉）——要看合/冲作用在谁身上、对用神有利还是有害。

## 四、速查

| 现象 | 典型含义 |
|---|---|
| 三合成局（用神所需） | 大助，事成力强 |
| 三合成局（忌神） | 大忌，合力为害 |
| 三合牵绊 | 事纠缠不散、合伙 |
| 六合（用世相合） | 缘深、亲近 |
| 六合绊住动爻 | 拖延、不动，待冲开 |
| 六冲 | 变动、散、破 |
