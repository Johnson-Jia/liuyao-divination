# 06 · 三合六合

"合"主和合、牵绊、聚集。合能把分散的力量聚成一股（三合成局），也能把爻绊住不动（合住）。合与冲相对：冲主动荡散开，合主聚集纠缠。

## 一、三合局

三合局由三个地支合成一股五行之力：

| 三合 | 化局 | 中神 |
|---|---|---|
| 申 · 子 · 辰 | 水局 | 子 |
| 亥 · 卯 · 未 | 木局 | 卯 |
| 寅 · 午 · 戌 | 火局 | 午 |
| 巳 · 酉 · 丑 | 金局 | 酉 |

（每局由"长生·帝旺·墓库"三支组成，中神为帝旺之支。）

### 成实局的条件（《增删卜易》主流严格标准）
三合要论为**实成之局**（合力大、能化、可作决定性力量），须满足以下任一：
1. **三爻齐动**：卦中三支皆为动爻。
2. **两动一静**：两动爻 + 一静爻（或以内/外卦动变凑合），且中神旺相。**一爻明动、一爻暗动，亦作两爻动**（暗动之爻可参与成局）。
3. **内外卦动变凑局**：通过动变使三支到位。

**关键：单爻独动、仅借月建/日辰凑合者，不得论实局**，只能作下述"虚合待用"。

### ⭐ 虚合待用局（最易判错）
当三支中**只有一爻（或零爻）是卦内动爻**，其余靠**月建、日辰**（月日非卦内爻、本身不动）凑齐时：

- **不是实成三合局**，是"虚一待用"之合。
- 只能作**牵绊、纠缠之象**的辅助参考（合的意向在，故主有牵连、不易立刻断），**不可断"局已成、合化生世/克世"的决定性旺力**。
- 须**待流月/流日填实**（缺失之支当令、或卦中再有动爻加入）方能真正发力成局。
- 断长期格局时，**勿把虚合的牵绊力度放大**为实局的合力。

### 实成三合局的含义（须先满足成局条件）
- 合力**大而专一**，化出之五行力量强，相当于一个强大的该五行之爻。
- 主**聚集、合伙、牵绊、纠葛**；合局之五行若为用神所需则大吉，为忌神则大凶。
- 合局也主**缠绵、拖延**（合住之意）。

> **经典例（本项目卦例）**：用神亥水（五爻·静）+ 世爻卯木（二爻·动）+ 月建未土。三支亥卯未齐备，但**仅世爻卯一爻独动**，亥为静爻、未是月建（非卦内爻）。
>
> ⚠️ **据此只可论"虚一待用之合"**，不可断"三合木局已成、合力聚世"。正确表述：有合的牵绊意向（缘分支在、有牵连，不至立刻断绝），但未成实局，须待填实（如亥子月水生木、或再有动爻加入）方真正发力。化向为木（官鬼），即便成局也带压力，非纯益。

## 二、六合（地支六合）

相邻地支两两相合：

| 六合 | 化气（参考） |
|---|---|
| 子 · 丑 | 土 |
| 寅 · 亥 | 木 |
| 卯 · 戌 | 火 |
| 辰 · 酉 | 金 |
| 巳 · 申 | 水 |
| 午 · 未 | 火（或土） |

### 六合的两种作用
1. **合住（绊）**：爻被合住则**不能动**，主拖延、羁绊、停顿、纠缠不清。动爻被合则不能动弹（须冲开方动）。
2. **合起（助）**：衰爻逢合得助，或合化为旺相之五行而增力。

### 六合含义
- 主**和合、亲近、牵连**。
- 占姻缘见合为吉（主和合亲近）；占谋事见合住则主拖延不动。
- 用神与世爻相合 → 双方亲近、缘分深。
- 忌神与用神相合 → 忌神被绊住，暂不克用神，反吉（但合解则克复发）。

## 三、合与冲的实务

| | 主 | 含义 |
|---|---|---|
| 三合 | 聚集化一 | 合力大、专一、牵绊、合伙 |
| 六合 | 两两和合 | 亲近、绊住、拖延 |
| 冲（六冲） | 散开动荡 | 变动、冲突、散、破 |

- **合逢冲则散**：合局/六合遇冲则解开（合住之爻遇冲则能动了；合局遇冲则散）。
- **冲逢合则止**：相冲之爻遇合则冲突暂止。
- 占事喜合忌冲（多数情况下，合主成、冲主变），但**合未必皆吉**（合住用神反主迟滞），**冲未必皆凶**（冲开墓库、冲散忌神反吉）——要看合/冲作用在谁身上、对用神有利还是有害。

## 三·补、六合六法 / 六冲六法 / 爻冲五类

### 六合的六种情形（《增删卜易·六合章》）
1. **日月合爻**：如丑月占，世爻子水与月建作合。
2. **爻与爻合**：两动爻相合（须两爻皆动，一爻不动不为合）。
3. **爻动化合**：动爻化出之爻与本身作合（化扶）。
4. **卦逢六合**：内外六爻自相和合（六合卦，不动亦是）。
5. **六冲变六合**：六冲卦变六合卦。
6. **六合变六合**：六合卦变六合卦。

**合的四态**：静逢合＝**合起**（休囚逢合亦有旺意）；动逢合＝**合绊**（绊住不能动）；爻与爻合＝**合好**（他来合我、相助）；爻动化合＝**化扶**（得扶助）。

### 六冲的六种情形（《增删卜易·六冲章》）
1. **日月冲爻**。2. **卦逢六冲**（六冲卦）。3. **六合变六冲**。4. **六冲变六冲**。5. **动爻变冲**。6. **爻与爻冲**。

冲主**散**：占凶事宜冲散，占吉事不宜。须兼用神——用神旺虽冲不碍，用神失陷则凶上加凶。**六合卦变六冲，用神纵旺亦始吉终凶**（有头无尾）。

### 爻冲五类
- **月冲 ＝ 月破**。
- **静爻旺逢日冲 ＝ 暗动**。
- **休囚逢日冲 ＝ 日破**。
- **动爻自化回头冲 ＝ 如逢仇敌**。
- **爻与爻冲 ＝ 相击**。

## 四、速查

| 现象 | 典型含义 |
|---|---|
| 三合成局（用神所需） | 大助，事成力强 |
| 三合成局（忌神） | 大忌，合力为害 |
| 三合牵绊 | 事纠缠不散、合伙 |
| 六合（用世相合） | 缘深、亲近 |
| 六合绊住动爻 | 拖延、不动，待冲开 |
| 六冲 | 变动、散、破 |
