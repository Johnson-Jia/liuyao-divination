# 04 · 动静暗破

旺衰定了"有没有力"，本册讲"动不动、被不被破坏"——这是事情能否发动、有无阻碍的关键。**其中"暗动 vs 日破"是最易判错、也最影响结论的一处。**

## 一、动静

- **动爻（明动）**：占得老阳(☰⃝)/老阴(☓)之爻，主动变化，力量显明，主事情有变。动则有化（见 05）。
- **静爻**：少阳/少阴之爻，安静不动，被动受生克。静爻本不能主动生克他爻——**除非被日辰冲起而为"暗动"**。

## 二、月破

**月建所冲之爻为月破。**

地支相冲：子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲。

> 例：未月，未冲丑，故**丑为月破**；卯月，卯冲酉，酉为月破。

- 月破之爻**一月之内无力、衰败、被破坏**，主散、主败、主落空。
- 月破是**暂时**的——过月（出月）即恢复，不等于永久无用。
- 忌神月破为吉（阻力被破），用神/元神月破为凶（无力生扶）。
- 爻若本身旺相而被月冲，仍论月破，但因有根基，过月即复，损害较轻。

## 三、日破 与 暗动（核心）

**日辰所冲之静爻**，分两种命运，判据是该爻**旺衰**：

### ⭐ 暗动（旺相逢冲）
**静爻旺相 + 被日辰冲 = 暗动。**

- 旺相 = 得月建生扶/比和/进气（相、旺、长生、临官、帝旺、冠带等）。
- 暗动之爻**等同于发动**，能生克他爻，**有用**。
- "暗"主不知不覚、隐秘、暗中。暗动主**暗中发力、暗中有人相助**。
- **元神（生用神者）暗动** → 暗中生助用神，**吉**。
- **忌神（克用神者）暗动** → 暗中克损用神，**凶**。
- 判暗动时**务必先辨该爻是元神还是忌神**，方能定吉凶。

### 日破（衰逢冲）
**静爻休囚衰弱 + 被日辰冲 = 日破（冲散）。**

- 衰弱 = 月克月泄月耗、死绝病衰墓等退气。
- 日破之爻**被冲散、无用**，主败、主散、主不成。

### 判定流程
```
静爻被日冲 → 爻旺相吗？
                 是 → 暗动（有用，能生克他爻）
                 否（休囚衰）→ 日破（无用，冲散）
```

> **经典例（本项目验证卦例）**：子孙酉金，未月（未土生金，金相、冠带进气，旺相），逢卯日冲 → **暗动**，非日破。作为元神暗中生助妻财亥水（金生水），是卦中隐性托底之力——主长辈介绍人暗中撮合。**若误判为日破，结论将完全相反。**

## 四、空亡（旬空）

**以日辰查"旬空"**：日干支所属的那一旬中，不出现的两个地支即空亡。

### 旬空表
| 旬 | 空亡 |
|---|---|
| 甲子旬 | 戌、亥 |
| 甲戌旬 | 申、酉 |
| 甲申旬 | **午、未** |
| 甲午旬 | 辰、巳 |
| 甲辰旬 | 寅、卯 |
| 甲寅旬 | 子、丑 |

> 查法：日干支在哪一旬，该旬空两个地支。如辛卯日属甲申旬，空**午、未**。

### 空亡的论法
1. **日月建本身不以空论**——月建、日辰是主宰，不空。只看**卦中之爻**是否落空亡。
2. **卦中爻空**：
   - **旺相之爻空 = 假空**（旺不为真空），待出空/填实即有用。
   - **衰弱之爻空 = 真空/空而不实**，主虚浮、落空、不实、不成。
   - **真空的具体季节**（《增删卜易》）：**春土、夏金、秋木、冬火**为真空——当季处死地又落旬空，到底全空；四季月（辰戌丑未，土当令）无真空。例：秋月（申酉）木爻落空为真空，冬月（亥子）火爻落空为真空。引擎以 `is_true_void` 判定。
3. **动而化空**：动爻变出空亡之爻，主事将成而落空，或暂时不能落实，待填实。
4. **空而逢冲 / 逢合 / 填实**：空亡之爻遇冲（冲空）、遇同地支之日（填实/出空）则空解，恢复作用。
5. 用神空：所占之事当下不实、未到位、暂虚；忌神空则祸未发，反吉。

> **实务要点**：先查**卦中六爻地支是否落旬空**。若卦中无落空之爻，则空亡不构成主要矛盾——这是常被忽略却很重要的一步（很多人见"旬空"就大谈空亡，实际卦中无爻落空，影响为零）。

## 五、速查：动静暗破汇总

| 状态 | 条件 | 含义 |
|---|---|---|
| 明动 | 占得老阳/老阴 | 显性变化，能生克他爻，有化 |
| 暗动 | 静爻旺相 + 日冲 | 隐性发动，有用，暗中发力 |
| 日破 | 静爻衰弱 + 日冲 | 冲散无用，主败 |
| 月破 | 月建所冲 | 一月内衰败无用，过月可复 |
| 空亡(假) | 旺爻落旬空 | 暂空，填实则有用 |
| 空亡(真) | 衰爻落旬空 | 虚浮不实，主落空 |
