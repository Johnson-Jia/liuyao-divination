# 08 · 断卦步骤（完整工作流）

本册把前七册的技法串成一条**可重复的断卦流水线**，并用本项目验证卦例（地泽临 → 地雷复，辛卯日）完整走查一遍。

## 一、八步流程

```
①排盘定盘 → ②取用神 → ③用神旺衰 → ④世爻状态
→ ⑤查动静暗破 → ⑥查化变合墓 → ⑦综合定性 → ⑧定应期
```

| 步 | 要回答的问题 | 关键技法 |
|---|---|---|
| ① 排盘 | 宫属、世应、六亲、纳甲、六神、动爻变卦 | 装卦（01）；**优先用引擎** |
| ② 取用神 | 所占之事对应哪个六亲？临不临应？ | 用神取用（02） |
| ③ 用神旺衰 | 用神在月日下有力还是无力？ | 旺相休囚死、十二长生（03） |
| ④ 世爻状态 | 求测者自身如何？（旺衰/入墓空破/动化） | 旺衰、墓库（03、07） |
| ⑤ 动静暗破 | 哪些爻动？哪些暗动/月破/日破/空？ | 动静暗破（04） |
| ⑥ 化变合墓 | 动爻怎么变？有无三合六合？谁入墓？ | 进退神化、合、墓（05、06、07） |
| ⑦ 综合定性 | 吉凶方向？有层次、有条件地给结论 | 全部综合 |
| ⑧ 定应期 | 何时转机/成败？ | 用神待令、出墓待冲（03、07） |

## 二、每步的关键提醒

- **①** 盘必须排对。动爻要找准（错一动爻全盘错）。用引擎核验。
- **②** 先定"占什么事"，再取用神。用神临应最清晰。
- **③** 记住"旺衰≠吉凶"——旺衰定力，生克定方向。且"贵≠旺"。
- **④** 世爻是"你自己"，世爻退弱/入墓/化退，自身就不济，再好的外缘也接不住。
- **⑤** **暗动 vs 日破务必看旺衰**——这是最高频的错误。空亡先查卦中爻是否落空，无则不论。
- **⑥** 化进化退看同五行地支；回头生克看变爻对动爻；合局看聚散；墓库看困束与出墓。
- **⑦** 结论要分层：本质（用神与世的生克方向）+ 当下（旺衰、阻碍）+ 趋势。不要一句吉凶了事。
- **⑧** 应期要有依据：衰待令、墓待冲、空待填。

## 三、完整走查：地泽临 → 地雷复（辛卯日，占家人介绍女缘是否旺己）

> 占时：丙午年 乙未月 辛卯日。男占女方（家人介绍），问是否旺己。

### ① 排盘定盘
坤宫二世卦，世在二，应在五。独九二发动。

| 爻 | 六神 | 六亲 | 纳甲 | 标注 |
|---|---|---|---|---|
| 上六 | 螣蛇 | 子孙 | 癸酉金 | 静 |
| 六五 | 勾陈 | 妻财 | 癸亥水 | 应 |
| 六四 | 朱雀 | 兄弟 | 癸丑土 | 静 |
| 六三 | 青龙 | 兄弟 | 丁丑土 | 静 |
| 九二 | 玄武 | 官鬼 | 丁卯木 | 世·动·化庚寅 |
| 初九 | 白虎 | 父母 | 丁巳火 | 静 |

- 月建未、日辰卯、旬空午未。
- 动爻九二（阳）变阴，下卦兑→震，主卦临变复卦。

### ② 取用神
男占女方 → **妻财为用**。妻财癸亥水在五爻，**临应爻** → 用神与应重叠，指向明确，就是这个女性。

### ③ 用神旺衰
- 未月水**死**（受月建未土之克）+ 日辰卯木泄（亥生卯）→ **用神衰弱受压**。
- 但用神**临五爻君位**（贵）→ **贵而暂弱**：对象本身好、缘分分量重，但眼下能量没出来、偏淡。
- 用神**生世**（亥水生卯木）→ **本质旺我**，方向吉。

### ④ 世爻状态
世爻丁卯木，四重退守，**化退+入墓为双重衰败的主导**：
- 官鬼持世（忧虑负担）
- 入未墓（月建未为木墓，且未月木囚 → 月这一侧既衰又困束）
- 动化退神（卯→寅，**不可逆退气大势**，是核心病灶）
- 临玄武（暗自动摇）
- **日辰卯为木帝旺比和**：仅作**保命底**——保元气不致断绝、使世爻属"困中有根"而非"衰死"，但**不能扭转化退大势，也不负责挣脱墓锁**（出墓靠冲墓，见⑧）。勿抬高日辰救应权重。

### ⑤ 动静暗破
- **子孙酉金**：未月土生金、金相/冠带 → **旺相**；逢日辰卯冲 → **暗动**（非日破！）。作元神**暗中生助亥水** → 隐性托底，主长辈介绍人暗中撮合。
- **兄弟丑土**（六三、六四）：未月冲丑 → **月破**。忌神（克财）被破 → 外部竞争/破财阻力减弱（偏好）。
- **旬空午未**：卦中六爻无落空之爻 → 空亡不构成主要矛盾（卦象实在）。世爻入之未墓落空 → **空墓**，墓不牢，困束易脱。

### ⑥ 化变合墓
- 九二**化退神**（卯→寅）：世爻力量减退、退意（不可逆退势）。
- **三合**：亥（用神·静）·卯（世爻·动）·未（月建）。三支虽齐备，但**仅世爻一爻独动，亥静、未为月建非卦爻** → 属**虚一待用之合，非实成木局**。只可断"有牵绊意向、缘分支在不易立刻断"，**不可断"木局已成、合化聚世"**；须待填实（亥子月水生木等）方发力。

### ⑦ 综合定性
**本质旺你**（用神生世 + 临君位）——好缘分，对象不差且分量重。（三合为虚合待用，不作决定性旺力，仅作牵绊辅助参考。）
**当下不旺**：用神衰弱受压（感情能量弱、对方未发力）+ 世爻"化退+入墓"双重衰败为主导（日辰仅保命底）+ 子孙暗动暗中托底（隐性，不显）。
→ 缘分实而时机未至、有底子却暂封存；退势为主、困中有根。宜守宜养，不宜急进亦不宜放弃。

### ⑧ 定应期（须双向辩证）
- **感情（财亥水）最佳窗口**：**亥、子月**（初冬、仲冬，水当令）——财最旺、女方最有力，且世爻虽未出墓但有日辰根托底。**这是双方相对重叠的利好窗口。**
- **世爻出墓发力**：**丑月**（丑冲未开木墓）——利你自身状态回勇。**但注意错位**：丑月土当令克水、水气将退，财爻此时已转弱 → "你行她退"。
- 故勿笼统说"丑月发力即好转"；**真正要把握的是亥子月**。卦变复，主慢、重新蓄力、否极泰来（七日来复）。

### 结论
好缘分、能旺你、对象不差且分量重；当下"贵而暂弱"、元神暗动暗中托底、世爻"化退+入墓"双重退守（日辰保底、非解药）——缘分实而时机未至、退势为主。**最佳窗口在亥子月**（财旺、双方较有利）；丑月你出墓但财已转弱，须防时间错位。

## 四、输出规范

一份合格的断卦输出应包含：

1. **排盘**（表）+ 关键参数（月日空用神）。
2. **本质判断**：用神与世爻的生克关系定大方向。
3. **当下分析**：旺衰、动静暗破、化变合墓，逐项点明，**关键辨析处（暗动/空墓/进退）要讲清依据**。
4. **综合定性**：分层、有条件的结论，不一刀切。
5. **应期**：有依据的时间窗口。
6. **理性提醒**：术数非科学，结论供参考。

## 五、理性提醒

六爻为传统文化与占卜术，断的是"势、象、时机"，非事实预测。任何决策（感情、财务、健康等）须以现实理性判断为准，占卜仅作心态与时机之参考。
