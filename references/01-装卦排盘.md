# 01 · 装卦排盘

排盘是断卦的根基。盘排错，后面全错。本册讲如何把一个卦"装"完整：定宫、定世应、纳甲、配六亲、配六神、找动爻变爻。

> 实践中**优先用引擎排盘**（`python src/paipan.py …`），手工装卦仅作理解与核验。

| 卦 | 符 | 自然 | 五行 | 方位 | 阴阳 | 爻象(自下而上) |
|---|---|---|---|---|---|---|
| 乾 | ☰ | 天 | 金 | 西北 | 阳 | 阳阳阳 |
| 兑 | ☱ | 泽 | 金 | 西 | 阴 | 阳阳阴 |
| 离 | ☲ | 火 | 火 | 南 | 阴 | 阳阴阳 |
| 震 | ☳ | 雷 | 木 | 东 | 阳 | 阳阴阴 |
| 巽 | ☴ | 风 | 木 | 东南 | 阴 | 阴阳阳 |
| 坎 | ☵ | 水 | 水 | 北 | 阳 | 阴阳阴 |

记忆：乾三连、坤六断、震仰盂、艮覆碗、离中虚、坎中满、兑上缺、巽下断。

## 二、八宫与五行

六十四卦分属八宫，每宫以一个八纯卦为本宫，**宫的五行即本宫卦的五行**：

| 宫 | 本宫卦 | 五行 |
|---|---|---|
| 乾宫 | 乾为天 | 金 |
| 兑宫 | 兑为泽 | 金 |
| 离宫 | 离为火 | 火 |
| 震宫 | 震为雷 | 木 |
| 巽宫 | 巽为风 | 木 |
| 坎宫 | 坎为水 | 水 |
| 艮宫 | 艮为山 | 土 |
| 坤宫 | 坤为地 | 土 |

每宫八卦由本宫卦"变爻"而来，顺序固定：

1. **本宫卦**（八纯）
2. **一世**：初爻变
3. **二世**：初、二爻变
4. **三世**：初、二、三爻变
5. **四世**：初至四爻变
6. **五世**：初至五爻变
7. **游魂卦**：四爻变回本宫
8. **归魂卦**：下卦恢复本宫

**示范（乾宫）**：乾为天 →(初变) 天风姤 →(二变) 天山遁 →(三变) 天地否 →(四变) 风地观 →(五变) 山地剥 →(四回) 火地晋(游魂) →(下回) 火天大有(归魂)。

> 查任意一卦属何宫、何世，最稳妥用引擎：`python src/paipan.py --hexagram 地泽临 …` 会输出宫属与世爻。手工核对可按上法逆推。

## 三、世应

**世爻**代表求测者自身（或所测主体），**应爻**代表对方/所测之事。世应相隔两爻（隔三位）。

**世爻口诀**：

| 卦类 | 世爻位置 |
|---|---|
| 八纯（六世） | 上爻（六爻） |
| 一世 | 初爻 |
| 二世 | 二爻 |
| 三世 | 三爻 |
| 四世 | 四爻 |
| 五世 | 五爻 |
| 游魂 | 四爻 |
| 归魂 | 三爻 |

应爻 = 世爻隔两爻（世初→应四，世二→应五，世三→应上，世四→应初，世五→应二，世上→应三）。

> 例：地泽临为坤宫二世卦，**世在二爻，应在五爻**。

## 四、纳甲（天干地支）

每个卦的上下两个八卦各纳一个天干，六爻再配地支。

**八卦纳干**：

| 八卦 | 内卦(下) | 外卦(上) |
|---|---|---|
| 乾 | 甲 | 壬 |
| 坤 | 乙 | 癸 |
| 坎 | 戊 | 戊 |
| 离 | 己 | 己 |
| 震 | 庚 | 庚 |
| 巽 | 辛 | 辛 |
| 艮 | 丙 | 丙 |
| 兑 | 丁 | 丁 |

**地支配法**：阳卦（乾坎艮震）地支**顺行**，阴卦（坤巽离兑）地支**逆行**；阳卦配阳支序、阴卦配阴支序。完整六爻纳甲表：

| 八卦 | 内卦(初/二/三) | 外卦(四/五/上) |
|---|---|---|
| 乾 | 甲子 甲寅 甲辰 | 壬午 壬申 壬戌 |
| 坤 | 乙未 乙巳 乙卯 | 癸丑 癸亥 癸酉 |
| 坎 | 戊寅 戊辰 戊午 | 戊申 戊戌 戊子 |
| 离 | 己卯 己丑 己亥 | 己酉 己未 己巳 |
| 震 | 庚子 庚寅 庚辰 | 庚午 庚申 庚戌 |
| 巽 | 辛丑 辛亥 辛酉 | 辛未 辛巳 辛卯 |
| 艮 | 丙辰 丙午 丙申 | 丙戌 丙子 丙寅 |
| 兑 | 丁巳 丁卯 丁丑 | 丁亥 丁酉 丁未 |

> 例：地泽临 = 上坤下兑。下兑内卦：初九丁巳、九二丁卯、六三丁丑；上坤外卦：六四癸丑、六五癸亥、上六癸酉。

## 五、六亲

以**本宫五行**为"我"，定六亲：

| 关系 | 六亲 |
|---|---|
| 生我者 | 父母 |
| 我生者 | 子孙 |
| 克我者 | 官鬼 |
| 我克者 | 妻财 |
| 比和者（同我） | 兄弟 |

**示范（坤宫，我=土）**：火生土→父母；土生金→子孙；木克土→官鬼；土克水→妻财；土同土→兄弟。

故地泽临（坤宫）六亲：丁巳火=父母、丁卯木=官鬼、丁丑土=兄弟、癸亥水=妻财、癸酉金=子孙。

## 六、六神（神煞）

以**日干**起，六神固定顺序为：青龙→朱雀→勾陈→螣蛇→白虎→玄武，从**初爻**起逐爻上行。

**日干起例**：

| 日干 | 初爻起 |
|---|---|
| 甲、乙 | 青龙 |
| 丙、丁 | 朱雀 |
| 戊 | 勾陈 |
| 己 | 螣蛇 |
| 庚、辛 | 白虎 |
| 壬、癸 | 玄武 |

六神性情：青龙主喜庆桃花；朱雀主口舌是非消息；勾陈主拖延牵缠旧事；螣蛇主虚惊怪异纠缠；白虎主血光凶险刚猛；玄武主暗昧隐私欺骗。

> 例：辛卯日，庚辛起白虎于初爻。地泽临：初九白虎、九二玄武、六三青龙、六四朱雀、六五勾陈、上六螣蛇。

## 七、动爻与变卦

- **动爻**：占卦时老阳（⃝）或老阴（✕）之爻，主变化。阳动变阴，阴动变阳。
- **变卦**：动爻阴阳改变后所得的新卦。一卦可有多个动爻，皆变。
- 动爻之"变"产生：化进化退、化空化破、化回头生克、化墓等（见 `05-进退神化.md`）。

> 例：地泽临九二（阳）动，变阴，下卦兑(☱)变震(☳)，全卦成地雷复。九二丁卯动化复卦六二庚寅。
