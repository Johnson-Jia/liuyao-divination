# 07 · 墓库论

"墓库"是爻的一种特殊状态——力量被封存、被收藏、暂不能发挥作用。本次项目验证卦例中，世爻"入未墓"是核心病灶之一；而"入空墓"与"出墓"的细节，是常被忽略却关键的判读点。

## 一、五行墓库

每个五行都有一个"墓库"地支（辰戌丑未四库）：

| 五行 | 墓库 | 三合局印证 |
|---|---|---|
| 木 | **未** | 亥卯**未** |
| 火 | **戌** | 寅午**戌** |
| 金 | **丑** | 巳酉**丑** |
| 水 | **辰** | 申子**辰** |
| 土 | 辰 | （土寄水库） |

> 记忆：墓库就是该五行三合局的"库"那一位。

> ⚠️ **墓库配属切勿搞混**（断应期、入墓时常错处）：
> - 水 → **辰**（不是丑！丑是金墓）
> - 金 → **丑**（不是辰）
> - 木 → 未，火 → 戌，土 → 辰
> 即：**亥水/子水之墓在辰**；申金/酉金之墓在丑。冲某五行之墓，要先查对该墓库地支，再论冲墓出墓的双向影响（冲墓之支可能同时克泄用神，见下"出墓的双向"）。

## 二、爻如何"入墓"

一个爻的五行，其墓库地支若出现在以下位置，则该爻**入墓**：

1. **入月墓**：月建地支为该爻墓库。
2. **入日墓**：日辰地支为该爻墓库。
3. **入动墓**：卦中动爻之地支为该爻墓库。
4. **入化墓（自化墓）**：动爻变出之爻为本身墓库。

> **经典例**：世爻卯木，木墓在未；月建为未 → **世爻入月墓（未墓）**。

## 三、墓库的两面：库 vs 墓

| 爻的状态 | 墓库的性质 | 含义 |
|---|---|---|
| 爻**旺相**入墓 | 为"**库**"（库藏） | 有归宿、有储藏，是收束归藏，未必凶；旺者入库如宝入库 |
| 爻**衰弱**入墓 | 为"**墓**"（坟墓） | 被困束、衰败、力量被封存、暂不出头 |

实务中多统称"入墓"，主：**困束、收藏、施展不开、暂不能发挥作用、力量被封存**。

## 四、关键辨析

### 1. 墓库是"收藏"，不是"生扶"
某五行之墓库（如木墓未），其本身五行仍属土。**土不生木（木克土）**。故"未土对木"是**墓库收藏**关系，**不是生扶助旺**——生扶木的是水（水生木）。勿把墓库当生扶。

### 2. ⭐ 空墓（墓库落空亡）— 易脱之困
若墓库地支**恰落旬空**（如未墓，而未落空亡），则：

- **墓不牢、墓是虚的** → 困束带"虚惊"性质，**容易挣脱**，非死局。
- 这与"爻本身有强根（如得日辰比和）"互相呼应：**困是暂困、可解之困**。

> **经典例**：世爻卯木入未墓，而未恰落旬空（辛卯日甲申旬空午未）→ 世爻入的是**空墓**，墓不甚牢，困束带虚、易脱。故断"暂困可解"，非衰败死局。

### 3. ⭐ 出墓（待冲）
入墓之爻**不能自己出来**，要靠**冲墓**（冲开墓库）方能出墓、重新发挥作用：

- 冲墓 = 用与墓库相冲的地支。如入未墓，待**丑**（丑未冲）冲开。
- 出墓的时机：冲墓之**月**、之**日**，或卦中有力的冲墓之**动爻**。

> **经典例**：世爻入未墓，待**丑月**（丑冲未）冲开墓库，世爻方能出墓、状态回勇、能主动发力。这与"用神待令"的应期常叠合。

### ⭐ 冲墓的双向影响（应期辩证，勿只看利好）
冲墓出墓虽利"入墓之爻"解脱，但**冲墓之支本身对其他爻（尤其用神）可能同时不利**，须双向看：

- 冲未墓用的是**丑**。丑属土，**土克水**；且丑为季冬末、水气将退。
- 故本例中，丑月虽开木墓让世爻（卯木）出墓，但对**妻财亥水**（财/她）是"土克水 + 水气衰退" → **财爻在丑月转弱**。
- 结果是**时间错位**：财（感情）最旺在**亥、子月**（水当令），世爻出墓发力却偏在**丑月** → "你状态回来时，她/感情能量已过峰值"。断应期须点明此错位，**最佳重叠窗口在亥子月**（财旺，世爻虽未出墓但有日辰根托底），而非笼统说"丑月发力"。

> 通则：**冲墓之支若同时克泄用神，则"出墓利好"与"用神转弱"并存**，应期当取用神得令且墓可冲的交集，而非单看一处。

## 五、入墓的吉凶

入墓本身是"力量被封存"的中性状态，吉凶看：

1. **谁入墓**：
   - 用神/世爻入墓 → 所占之事或自身被困束、暂难发挥（偏负）。
   - 忌神入墓 → 忌神被收藏不能为害（偏正）。
2. **旺衰**：旺入库（有藏有出），衰入墓（衰败被困）。
3. **是否空墓**：空墓易脱，损害轻、可解。
4. **能否出墓**：有冲墓之机则终能出，事有转机；无则长期困顿。

## 六、速查

| 状态 | 含义 |
|---|---|
| 旺爻入墓 | 入库，归藏，有出 |
| 衰爻入墓 | 入墓，困束，衰败 |
| 入空墓 | 墓虚，困束易脱，非死局 |
| 待冲出墓 | 冲墓之月/日出墓发力 |
| 忌神入墓 | 忌神不能为害，反吉 |
