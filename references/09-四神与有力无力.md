# 09 · 用神 · 元神 · 忌神 · 仇神 · 有力无力

断卦以**用神**为核心，但用神能否得生、是否被克，要看**元神、忌神是否有力**。"见生不生""虽动不克"是六爻里极常见、也极致命的误判点——元神现了但无力，照样生不起用神；忌神动了但无力，照样克不到用神。本章把这套判定讲透。

> 依据：《增删卜易·用神元神忌神仇神章》。

## 一、四神定义（以用神爻五行为基准）

| 神 | 定义 | 对用神的作用 |
|---|---|---|
| **用神** | 所占之事对应的爻 | 主体 |
| **元神** | 生用神者（生我） | 生扶用神，吉 |
| **忌神** | 克用神者（克我） | 克损耗用神，凶 |
| **仇神** | 克元神、生忌神者 | 间接害用（制元助忌） |
| 闲神 | 与用神无直接生克 | 不主事 |

> 例：用神=金 → 元=土（生金）、忌=火（克金）、仇=木（克土元、生火忌）。
> 注意以**用神爻的五行**为基准（非六亲），方符合"元生用、忌克用、仇克元生忌"的定义。

## 二、⭐ 元神有力五条（能生用神，诸占皆吉）

1. 元神**旺相**，或临日月，或得日月动爻生扶。
2. 元神**动化回头生**、及**化进神**。
3. 元神**长生、帝旺于日辰**。
4. 元神**与忌神同动**（忌动则元亦动以制之，反成生用之势）。
5. 元神**旺动，临空、化空**（旺不为空，待出空即生）。

## 三、无用元神六条（虽现，不能生用，虽有如无）

1. 元神**休囚不动**，或动而休囚又被伤克。
2. 元神休囚，又逢**旬空、月破**。
3. 元神**动化退神**。
4. 元神**衰而又绝**。
5. 元神**入三墓**。
6. 元神休囚，**动化绝、化克、化破、化散**。

> 要害：**"见生不生"**——元神现了但命中上述任一，则生不起用神，断吉要打折扣。

## 四、忌神有力五条（能克用神，势同斧钺，诸占大凶）

1. 忌神旺相，或遇日月动爻生扶，或临日月。
2. 忌神动化回头生、化进神。
3. 忌神旺动，临空、化空。
4. 忌神长生、帝旺于日辰。
5. 忌神与仇神同动。

## 五、忌神动不克用神七条（虽动，不能克用，化凶为吉）

1. 忌神休囚不动、动而休囚被日月动爻克。
2. 忌神静，临空破。
3. 忌神入三墓。
4. 忌神衰，动化退神。
5. 忌神衰而又绝。
6. 忌神动，化绝、化克、化破、化散。
7. **忌神与元神同动**（元神同动则制忌护用，忌神克力被化解）。

## 六、用神有根是前提

> **用神无根，谓之"元神有力亦难生、忌神无力亦休喜"。**

用神本身要有气（得月生扶、临日月、有动等）。若用神自身衰绝无根，则：元神再有力也"生不起一个死物"；忌神虽无力，用神自己也立不住，未必就能吉。故断卦先看**用神有没有根**，再看元忌有力无力。

## 七、仇神的作用

仇神不直接生克用神，但它**克元神**（使元神无力生用）、**生忌神**（助忌神克用），是用神的间接大敌。断卦见仇神旺动，要警惕"元神被制、忌神得助"的连锁。

## 八、引擎支持

- `board.line.role` 返回四神角色（用/元/忌/仇/闲）。
- 元神、忌神爻另有 `line.effective`（True 有力 / False 无力 / None 中和）与 `line.effective_reason`（理由），按上述有力/无力条件自动判定。
- `board.yongshen_wx` 为用神爻五行基准。

> 经典例（本项目卦例）：用神妻财亥水。
> - 子孙酉金 = 元神，**旺相暗动 → effective=True「发动有力」** → 能暗中生财（长辈撮合）。
> - 兄弟丑土 = 忌神，但**月破 → effective=False「月破无力」** → 虽是忌神却克不到财（外部阻力被破）。
> - 父母巳火 = **仇神**（克金元、生土忌）。
> 这正是"元神有力则生、忌神无力则不克"的活用——若无这套判定，易误判"有忌神就凶"。
