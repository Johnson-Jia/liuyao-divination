# 02 · 用神取用

**用神**是全卦核心——所占之事对应的那个六亲爻。取对用神，断卦方向才对；取错，旺衰断得再细也白费。

## 一、六亲对应的占事

| 六亲 | 主占之事 |
|---|---|
| **父母** | 父母长辈、祖业、文书、契约、学历、考试、房屋、土地、车辆、舟车、衣物、雨、辛劳 |
| **兄弟** | 兄弟姐妹、同辈朋友、合伙人、竞争者、劫财破财之事、同僚 |
| **子孙** | 子女晚辈、下属、学生、医药（治病看子孙）、福神、僧道、宠物、六畜、娱乐、解忧之神（子孙克制官鬼，主消灾） |
| **妻财** | 妻子、女友、钱财、财物、粮食、收成、佣人、晴（测天看财为晴） |
| **官鬼** | 丈夫、男友、官职、工作、功名、领导、盗贼、忧虑、疾病、官非、雷霆、雷电 |

## 二、常用取法

| 占问 | 用神 |
|---|---|
| 占父母/长辈吉凶 | 父母 |
| 占子女/晚辈 | 子孙 |
| 占夫妻（男占妻、女占夫） | 男看妻财，女看官鬼 |
| 占恋爱对象（男占女/女占男） | 同上：男看妻财，女看官鬼 |
| 占财运、求财、生意 | 妻财（辅看子孙为财源） |
| 占考试、求职、升迁、官运 | 官鬼（功名）+ 父母（文书） |
| 占疾病 | 官鬼为病（也看子孙为医药、为解神） |
| 占官司、官非 | 官鬼 |
| 占失物、盗贼 | 官鬼（贼）+ 妻财（物） |
| 占行人、出行 | 看应爻或用神（行人之爻视关系定） |
| 占合伙、竞争 | 兄弟 |
| 占盖房、买房 | 父母（房屋） |
| 占他人、泛占某事无特定六亲 | 应爻 |

## 三、用神与世、应的关系

- **世爻** = 求测者自己（自占吉凶以世爻为主体）。
- **应爻** = 对方 / 所测的对象或事。
- **用神临应爻**：用神与应爻重叠，指向明确，是好事——说明所测之人/事与用神一致。
- **用神与世爻的生克**决定吉凶方向：
  - 用神**生世**、**合世** → 有利、旺我、吉。
  - 用神**克世**、**冲世** → 耗损、压力、不吉（除非占官鬼求官等用神本宜克我之类）。
  - 世爻**生用神** → 自己付出、费力。
  - 世爻**克用神** → 自己克制所测之物。

## 四、用神不上卦（伏神）

若所占用神在主卦六爻中**没有出现**（不在卦上），称为"用神不上卦"，需寻**伏神**：

- 在本宫八纯卦（首卦）中找到该六亲所在之爻，即为伏神，伏于主卦某爻之下。
- 飞神 = 主卦上承载伏神的那个爻。
- 伏神要"得出"（飞神空、破、被克，或日月生扶伏神）才能成用；若伏神被飞神压制、且衰弱无气，则所求难遂。

> 实务中用神不上卦多主事情不顺、所求之人或事目前缺位或难现。引擎目前以显用神为主；伏神需手工详参。

## 五、取用神的关键提醒

1. **先定占什么事，再定用神**——同一卦，问财与问病用神不同。
2. **用神取定后，全卦围绕它转**：旺衰看它、生克看它、应期看它。
3. **勿把应爻当用神混用**——应爻是"对方这个位置"，用神是"所占之事对应的六亲"。二者重合时最清晰，不重合时要分清（比如男占女友，用神是妻财爻，而应爻可能是女友也可能是别人，以妻财为准）。
4. 占**自身整体吉凶**、**谋事成败**无特定对象时，以**世爻**为主来看。
